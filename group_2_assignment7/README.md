# Group 2 - Assignment 7: Sky Dodge

## How to Run
1. Open `group_2_assignment7/group_2_assignment7.sln` in an IDE.
2. Make sure .NET 9.0 and MonoGame 3.8 are installed.
3. Click Run/Start to build and launch the game.

---

## How to Play
- Use **Left / Right Arrow Keys** to move the player.
- Avoid falling swords (obstacles).
- Collect falling stars to gain points.
- Reach **50 points to win**.
- Lose all **3 lives to lose the game**.

---

## Controls
- **Left Arrow**: Move left
- **Right Arrow**: Move right
- **Enter**: Start game
- **P**: Pause / Resume game
- **R**: Restart after win/lose

---

## Game Features
- Player movement with keyboard input
- Falling obstacles and collectibles
- Increasing difficulty over time
- Collision detection system
- HUD showing score, lives, and difficulty
- Title, pause, win, and lose screens
